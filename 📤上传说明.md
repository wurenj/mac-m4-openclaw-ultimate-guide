# 📤 如何上传本教程到 GitHub

> 由于 GitHub 需要交互式认证，请按以下步骤手动上传

---

## 方法 1：使用 GitHub Desktop（最简单）

### 步骤 1：下载 GitHub Desktop
```
访问：https://desktop.github.com
下载并安装
```

### 步骤 2：登录 GitHub
```
1. 打开 GitHub Desktop
2. 使用你的 GitHub 账号登录
3. 账号：wurenj
```

### 步骤 3：添加本地仓库
```
1. File > Add Local Repository
2. 选择文件夹：/home/admin/.openclaw/workspace/github/mac-m4-openclaw-ultimate-guide
3. 点击 "Add Repository"
```

### 步骤 4：发布到 GitHub
```
1. 点击右上角 "Publish repository"
2. 名称：mac-m4-openclaw-ultimate-guide
3. 描述：Mac mini M4 + OpenClaw 超详细部署指南（15 万字）
4. 勾选 "Keep this code private"（如果想私有）
5. 点击 "Publish"
```

## 方法 2：使用命令行（需要 Personal Access Token）

### 步骤 1：生成 Personal Access Token
```
1. 访问：https://github.com/settings/tokens
2. 点击 "Generate new token (classic)"
3. 填写说明：Mac Upload
4. 勾选 scopes：repo, gist
5. 点击 "Generate token"
6. 复制生成的 token（只显示一次，保存好）
```

### 步骤 2：上传教程
```bash
# 进入教程目录
cd /home/admin/.openclaw/workspace/github/mac-m4-openclaw-ultimate-guide

# 添加远程仓库（替换 YOUR_TOKEN 为你的 token）
git remote add origin https://YOUR_TOKEN@github.com/wurenj/mac-m4-openclaw-ultimate-guide.git

# 推送
git push -u origin main
```

## 方法 3：网页直接上传

### 步骤 1：创建 GitHub 仓库
```
1. 访问：https://github.com/new
2. 仓库名：mac-m4-openclaw-ultimate-guide
3. 描述：Mac mini M4 + OpenClaw 超详细部署指南
4. 选择 Public 或 Private
5. 不要勾选 "Initialize with README"
6. 点击 "Create repository"
```

### 步骤 2：压缩教程文件
```bash
# 在终端运行
cd /home/admin/.openclaw/workspace/github/
zip -r mac-m4-openclaw-ultimate-guide.zip mac-m4-openclaw-ultimate-guide/ -x "*.git*"
```

### 步骤 3：上传到 GitHub
```
1. 打开刚创建的仓库页面
2. 点击 "uploading an existing file"
3. 把压缩文件拖拽到浏览器
4. 等待上传完成
5. 点击 "Commit changes"
```
