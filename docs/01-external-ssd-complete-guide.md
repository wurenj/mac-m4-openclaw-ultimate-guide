## 1.1 为什么要用外置 SSD 作为系统盘？

### 你的设备情况

```
Mac mini M4 配置：
├── 处理器：Apple M4 (10 核 CPU + 10 核 GPU + 16 核神经网络引擎)
├── 内存：16GB 统一内存
├── 内置存储：256GB SSD ❌ 太小
└── 外置存储：致态 2TB PSSD ✅ 主力使用
```

### 256GB 内置硬盘的实际可用空间

```
macOS 系统占用：约 30GB
系统预留空间：约 20GB（用于缓存、交换文件）
实际可用：约 206GB

安装以下软件后：
├── Office 套件：~3GB
├── 开发工具：~5GB
├── 常用应用：~10GB
├── 照片/音乐/视频：~50GB（如果你存储的话）
└── 剩余空间：~138GB ❌ 不够用！
```

### 外置 SSD 方案优势

| 优势 | 说明 |
|------|------|
| ✅ **空间充足** | 2TB 足够装系统 + 所有软件 + 模型 + 项目文件 |
| ✅ **性能优秀** | USB 3.2 1000MB/s 足够流畅运行 macOS |
| ✅ **灵活便携** | 系统随身携带，即插即用 |
| ✅ **安全备份** | 内置硬盘可作为额外备份空间 |
| ✅ **成本低** | 比买 512GB/1TB 内置版便宜很多 |

### 性能对比测试

```
内置 256GB SSD：
├── 顺序读取：~2800 MB/s
├── 顺序写入：~1500 MB/s
└── 实际体验：启动快 5-8 秒

外置 致态 2TB PSSD（USB 3.2）：
├── 顺序读取：~1000 MB/s
├── 顺序写入：~800 MB/s
└── 实际体验：日常使用几乎无差别

结论：外置 SSD 完全够用！✅
```

```
打开包装盒，检查以下物品：

□ Mac mini M4 主机
□ 电源线（编织材质，质量很好）
□ USB-C 转 Lightning 连接线（用于初始设置）
□ Apple 贴纸
□ 快速入门指南
□ 保修卡

检查主机外观：
□ 无划痕、无凹陷
□ 接口无损坏
□ 底部橡胶垫完好
```

### 步骤 2：检查致态 2TB PSSD

```
打开包装，检查：

□ SSD 主体
□ USB-C to C 数据线
□ USB-C to A 转接头
□ 说明书
□ 保修卡

检查 SSD 外观：
□ 外壳无划痕
□ 接口无损坏
□ 指示灯正常
```

### 步骤 3：准备配件

**必需配件：**
```
□ 稳定的电源插座（不要使用劣质插线板）
□ 有线网络（推荐）或 Wi-Fi
□ 显示器（通过 HDMI 或 USB-C）
□ 键盘和鼠标（蓝牙或有线）
```

**推荐配件：**
```
□ 高质量 USB-C 数据线（如果原装线不够长）
□ USB 集线器（扩展更多接口）
□ 散热支架（Mac mini 放在通风处）
```

### 步骤 4：连接设备

```
1. 连接电源：
   └── 将电源线插入 Mac mini 背部电源接口
   └── 另一端插入稳定的电源插座

2. 连接显示器：
   ├── HDMI 连接：HDMI 线 → Mac mini HDMI 口 → 显示器
   └── USB-C 连接：USB-C 线 → Mac mini USB-C 口 → 显示器

3. 连接键盘鼠标：
   ├── 蓝牙设备：开机后自动配对
   └── 有线设备：插入 USB-A 或 USB-C 口

4. 连接网络：
   ├── 有线网络（推荐）：网线 → Mac mini 以太网口 → 路由器
   └── Wi-Fi：首次开机后设置

5. 连接致态 SSD：
   └── USB-C 线 → SSD → Mac mini USB-C 口（推荐雷电 4 口）
```

## 1.4 格式化外置 SSD（关键步骤）

### 为什么要格式化？

```
新买的 SSD 通常是 exFAT 格式，不适合安装 macOS。
我们需要格式化为 APFS 格式（Apple 原生文件系统）。

⚠️ 警告：格式化会删除 SSD 上所有数据！
   新 SSD 是空的，所以可以直接格式化。
```

### 步骤 1：打开磁盘工具

```
1. 在恢复模式主菜单，选择"磁盘工具"
2. 点击"继续"
3. 等待磁盘工具加载
```

### 步骤 2：识别外置 SSD

```
磁盘工具左侧边栏会显示所有磁盘：

内置磁盘：
└── Apple SSD AP256Q (内置 256GB)
    └── Macintosh HD

外置磁盘（你的致态 SSD）：
└── ZhiTai PSSD USB Media (约 2TB)
    └── Untitled

⭐ 重要：选择外置 SSD（容量约 2TB 的那个）
   不要选错成内置硬盘！
```

**识别技巧：**
```
□ 看容量：外置 SSD 显示约 2TB (2000GB)
□ 看名称：通常显示"ZhiTai"、"PSSD"或"Untitled"
□ 看图标：外置磁盘有弹出符号 ⏏
□ 位置：在左侧边栏的"外置"分类下
```

### 步骤 3：格式化 SSD

**详细步骤：**

```
1. 在左侧边栏，选择外置 SSD 的**设备**（顶层，不是下面的卷）
   ⭐ 重要：选择设备名称（如"ZhiTai PSSD USB Media"），不是下面的"Untitled"

2. 点击顶部工具栏的"抹掉"按钮

3. 填写格式化信息：
   ┌─────────────────────────────────────┐
   │ 名称：MacOS-SSD                     │
   │ 格式：APFS                          │
   │ 方案：GUID 分区图                    │
   └─────────────────────────────────────┘

   详细说明：
   - 名称：可以自定义，建议用"MacOS-SSD"或"外置系统"
   - 格式：必须选择"APFS"（macOS 原生格式）
   - 方案：必须选择"GUID 分区图"（启动必需）

4. 点击"抹掉"按钮

5. 等待格式化完成（约 30-60 秒）
   - 看到进度条
   - 完成后显示"抹掉完成"

6. 点击"完成"

7. 点击左侧边栏的"MacOS-SSD"卷（刚才创建的新卷）
   - 确认格式化成功
   - 应该显示：APFS 格式，约 2TB 可用空间
```

**格式化成功标志：**
```
✓ 左侧边栏显示"MacOS-SSD"
✓ 右侧显示：APFS 格式
✓ 容量显示约 2TB
✓ 可用空间显示约 2TB
```

### 命令行格式化（可选，高级用户）

```bash
# 1. 列出所有磁盘
diskutil list

# 输出示例：
# /dev/disk0 (internal):
#    #:                       TYPE NAME                    SIZE
#    0:      GUID_partition_scheme                         256.0 GB
#    1:             Apple_APFS Container disk1             256.0 GB
#
# /dev/disk2 (external):
#    #:                       TYPE NAME                    SIZE
#    0:      GUID_partition_scheme                         2.0 TB
#    1:                 EFI EFI                           314.6 MB
#    2:               Apple_APFS MacOS-SSD                 2.0 TB

# 2. 找到外置 SSD（通常是 disk2 或 disk3）
# ⚠️ 务必确认容量正确（约 2TB）

# 3. 格式化（替换 diskX 为实际磁盘编号）
diskutil eraseDisk APFS "MacOS-SSD" /dev/disk2

# 4. 验证格式化结果
diskutil info /Volumes/MacOS-SSD
```

## 1.6 设置外置 SSD 为默认启动盘

### 方法 1：系统设置（推荐，最简单）

```
1. 启动到外置 SSD 上的 macOS 后
2. 点击左上角苹果菜单 ()
3. 选择"系统设置"
4. 在左侧边栏选择"通用"
5. 点击右侧的"启动磁盘"
6. 看到两个选项：
   - Macintosh HD（内置）
   - MacOS-SSD（外置）
7. 选择"MacOS-SSD"
8. 点击"重新启动"
9. 确认选择
10. Mac 会重启，以后都会从外置 SSD 启动
```

### 方法 2：启动时选择（临时）

```
1. 关机
2. 按住电源按钮不放
3. 看到"正在载入启动选项"
4. 选择"MacOS-SSD"
5. 按回车启动

⭐ 注意：这只是临时选择，下次开机还需要手动选择
```

### 方法 3：命令行（高级用户）

```bash
# 1. 列出可用启动磁盘
bless --info /

# 2. 设置外置 SSD 为启动磁盘
sudo bless --mount /Volumes/MacOS-SSD --setBoot

# 3. 验证
bless --info /

# 输出应该显示：
# Currently selected boot volume: /Volumes/MacOS-SSD
```

## 1.9 外置 SSD 性能优化

### 优化 1：禁用 Spotlight 索引（提升性能）

```bash
# 打开终端（应用程序 > 实用工具 > 终端）

# 将外置 SSD 加入 Spotlight 隐私列表
sudo mdutil -a -i off /Volumes/MacOS-SSD

# 验证
mdutil -a -i

# 如果以后需要搜索文件，可以重新开启：
sudo mdutil -a -i on /Volumes/MacOS-SSD
```

### 优化 2：禁用 Time Machine 自动备份到自身

```bash
# 排除外置 SSD 从 Time Machine 备份
sudo tmutil addexclusion /Volumes/MacOS-SSD

# 验证
tmutil isexcluded /Volumes/MacOS-SSD

# 应该显示：[Excluded] D /Volumes/MacOS-SSD
```

### 优化 3：监控 SSD 温度

```bash
# 安装 smartmontools
brew install smartmontools

# 检查 SSD 健康状态
smartctl -a /dev/disk2

# 监控温度（重要！）
watch -n 5 smartctl -A /dev/disk2 | grep Temperature

# 正常温度：30-50°C
# 如果超过 60°C，需要改善散热
```

### 优化 4：定期维护

```bash
# 每月运行一次磁盘工具 First Aid
1. 打开磁盘工具
2. 选择 MacOS-SSD
3. 点击"急救"
4. 点击"运行"
```

## 1.10 内置 SSD 用途规划

### 推荐分配方案

```
内置 256GB SSD 用途：
├── macOS 系统缓存 (~20GB)
├── 应用程序 (/Applications) (~30GB)
├── 用户主目录部分数据 (~50GB)
│   ├── 桌面
│   ├── 文档
│   └── 下载
├── Time Machine 本地快照 (~50GB)
└── 剩余空间备用 (~106GB)

外置 2TB SSD 用途：
├── macOS 系统 (~30GB)
├── OpenClaw 工作空间 (~50GB)
├── 本地 AI 模型 (~200-500GB)
├── 项目文件 (~200GB)
├── 媒体文件 (~500GB)
└── 备份和归档 (~500GB)
```

### 迁移大型文件夹到外置 SSD

```bash
# 1. 创建目标文件夹
mkdir -p /Volumes/MacOS-SSD/Users/admin

# 2. 复制大型文件夹
cp -R ~/Movies /Volumes/MacOS-SSD/Users/admin/
cp -R ~/Pictures /Volumes/MacOS-SSD/Users/admin/
cp -R ~/Music /Volumes/MacOS-SSD/Users/admin/

# 3. 重命名原文件夹（作为备份）
mv ~/Movies ~/Movies.backup
mv ~/Pictures ~/Pictures.backup
mv ~/Music ~/Music.backup

# 4. 创建软链接
ln -s /Volumes/MacOS-SSD/Users/admin/Movies ~/Movies
ln -s /Volumes/MacOS-SSD/Users/admin/Pictures ~/Pictures
ln -s /Volumes/MacOS-SSD/Users/admin/Music ~/Music

# 5. 验证
ls -la ~/Movies
# 应该显示：Movies -> /Volumes/MacOS-SSD/Users/admin/Movies

# 6. 使用正常后，删除备份
rm -rf ~/Movies.backup ~/Pictures.backup ~/Music.backup
```

## 1.13 故障排查手册

### 问题 1：无法从外置 SSD 启动

**症状：**
```
□ 开机后直接进入内置系统
□ 启动选项中没有外置 SSD
□ 显示禁止符号 🚫
```

**解决方案：**
```
步骤 1：检查连接
□ 重新插拔 USB-C 线
□ 尝试不同的 USB-C 端口
□ 确保 SSD 指示灯亮起

步骤 2：进入启动选项
□ 关机
□ 按住电源按钮不放
□ 看到启动选项后，选择 MacOS-SSD

步骤 3：重新设置启动磁盘
□ 启动到 MacOS-SSD
□ 系统设置 > 通用 > 启动磁盘
□ 重新选择 MacOS-SSD
□ 重启

步骤 4：如果还不行，重新安装
□ 进入恢复模式
□ 磁盘工具 > 格式化 MacOS-SSD
□ 重新安装 macOS
```

### 问题 2：启动速度慢

**症状：**
```
□ 启动时间超过 2 分钟
□ 看到苹果标志后很久才进入系统
```

**解决方案：**
```
1. 检查 USB 线缆
   □ 使用原装或高质量 USB-C 线
   □ 线缆不要太长（建议 1 米以内）

2. 检查端口
   □ 连接到雷电 4/USB4 端口
   □ 不要使用 USB 集线器

3. 检查 SSD 温度
   □ 用手摸 SSD，如果很烫，需要散热
   □ 改善通风，加散热片

4. 减少启动项
   □ 系统设置 > 通用 > 登录项
   □ 移除不必要的启动项
```

### 问题 3：系统运行卡顿

**症状：**
```
□ 打开应用很慢
□ 切换窗口卡顿
□ 鼠标指针延迟
```

**解决方案：**
```
1. 检查内存使用
   □ 打开活动监视器（应用程序 > 实用工具）
   □ 查看内存压力
   □ 如果超过 80%，关闭一些应用

2. 检查磁盘空间
   □ 苹果菜单 > 关于本机 > 存储
   □ 确保外置 SSD 剩余空间 > 10%
   □ 清理不必要的文件

3. 重启系统
   □ 苹果菜单 > 重新启动

4. 运行磁盘急救
   □ 磁盘工具 > 选择 MacOS-SSD > 急救
```

### 问题 4：USB 连接不稳定

**症状：**
```
□ SSD 偶尔断开连接
□ 系统提示"磁盘未正常推出"
□ 文件传输中断
```

**解决方案：**
```
1. 更换 USB 线缆
   □ 使用原装线或认证的高质量线缆

2. 尝试不同端口
   □ 换另一个 USB-C 口

3. 检查供电
   □ 某些 SSD 需要额外供电
   □ 使用带供电的 USB 集线器

4. 更新系统
   □ 确保 macOS 是最新版本
```

### 问题 5：安装过程中断

**症状：**
```
□ 下载 macOS 时中断
□ 安装过程中黑屏
□ 进度条卡住不动
```

**解决方案：**
```
1. 检查网络连接
   □ 确保网络稳定
   □ 使用有线网络更可靠

2. 检查电源
   □ 确保电源连接稳定
   □ 不要使用劣质插线板

3. 重新开始
   □ 强制关机（长按电源 10 秒）
   □ 重新进入恢复模式
   □ 重新安装

4. 如果多次失败
   □ 尝试不同时间（网络可能拥堵）
   □ 联系 Apple 支持
```
